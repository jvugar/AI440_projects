6. Overfitting

 

          The overfitting means the network fits to closely to a particular dataset, and can’t generalize future data and make accurate predictions. It occurs in practice when there are more trainable weights than training samples, and those weights fit one particular dataset closely. We can reduce overfitting by adding more data, so that our network is trained based on various data, instead of a particular one. We can also do it by reducing redundant weights, which reduces the complexity of our network as well as risk of overfitting. Finally, we can add regularization, which adds a penalty as weights increases, and thus we can get a model with small weights, which also prevents overfitting.

With L2 regularization

(n=20, lr=0.001, lambda=0.5)

          L2 regularization prevents overfitting by penalizing large weights, so that the neural network is encouraged to have small weights by using all of inputs a little rather than using some inputs too much, which reduces overfitting effectively. Before using L2 regularization, the model’s prediction accuracy is high, but may over fit the dataset. After using L2 regularization, the model’s prediction accuracy lowers a little, but risk of overfitting is reduced.
